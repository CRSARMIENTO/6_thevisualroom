Bash aliases and functions useful in OpenFOAM

Clean swap files in the directory

 clSwap() { 
     find . -name ".*swp" -type f -delete 
 }

Clean $FOAM_CASE folder

 del() {
     dir=`pwd`
     mv $dir/0 $dir/init
     ls | grep -e '[0-9]' | xargs rm -r
     mv $dir/init $dir/0
 }

Allwmake/Allwclean function

  • wmake solver
 mk() {
     if [ -f ./Allwmake ]
      then
          ./Allwmake
      else
          wmake
     fi
 }
  • wclean solver
 mc() {
     if [ -f ./Allwclean ]
      then
          ./Allwclean
      else
          wclean
     fi
 }
  • wmake library
 mkl() {
     if [ -f ./Allwmake ]
      then
          ./Allwmake
      else
          wmake libso
     fi
 }
  • wclean library
 mc() {
     if [ -f ./Allwclean ]
      then
          ./Allwclean
      else
          wclean libso
     fi
 }

Paraview

To avoid compiling new Paraview version with OpenFOAM we can use an alias that would do almost the same as ParaFoam does (as far as I can see). The only difference is that when a case is opened you have to select OpenFOAM from the list is given, so that ParaView new what to look in the case folder.

  • Alias a new version of Paraview
   alias paraview='$NEW_PARAVIEW_INST_DIR/bin/paraview'
  • Alias similar to ParaFoam command
   alias pf='touch ${PWD##*/}".OpenFOAM; paraview ./*.OpenFOAM &'
